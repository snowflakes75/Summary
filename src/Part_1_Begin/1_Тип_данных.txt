Переменная

Переменная - это элементарный контейнер для хранения информации
У каждой переменной должен быть тип данных

::: тип_данных имяПеременной = значение;

Тип данных характеризуется
    * диапазон возможных значений
    * допустимые операции. Наприме, конкатенация для строк
    * способ хранениях этих значений в памяти. Объекты в куче (HEAP)
// Стоит отметить. что класс тоже как тип данных

Существует два вида типов данных
    * Примитивный
    * Ссылочный
