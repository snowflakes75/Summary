Общие заметки по Коллекциям


Коллекции являеются "массивом на стероидах"
Позволяют удобно обрабатывать данные.
Они по своей сути являются структурами данных.

Выделяют 4 базовые операции с данными CRUD
*create - создание
*read - чтение
*update - замена/обновление
*delete - удаление

Общий вид создание Коллекции

::: интерфейс/реализация имя_коллекции = new необходимая_реализация()

Начиная с Java 1.5 было введено обобщение, чтобы контроливовать тип,
с которым будет работать наша Коллекция

::: ArrayList<тип_данных> list = new ArrayList<>();
