Переменная

Переменная - это элементарный контейнер для хранения информации.
У каждой переменной должен быть тип данных.

Схема создание переменной
тип_данных имя_переменной = значение;

Тип данных характеризует:

  • Диапазон возможных значений
  • Допустимые операции. Например, конкатенация для строк
  • Способ хранения этих значений в памяти. Объекты в куче (HEAP)
Существует два вида тип_данных
  • Примитивный
  • Ссылочный
